# Image Commands

>**Kitsune**

* Get a random kitsune (foxgirl) image from nekos.life

    Example: `j!kitsune`

>**Wallpaper**

* Need a wallpaper? Get a random wallpaper from Jeanne

    Example: `j!wallpaper`

>**Jeanne**

* Get a random Jeanne d'Arc image from Jeanne

    Exapmle: `j!jeanne`

>**Saber**

* Get a random Saber image from Jeanne

    Example: `j!saber`
